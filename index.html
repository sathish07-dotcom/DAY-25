Object in js

In JavaScript, an **object** is a data structure used to store collections of data in the form of **key-value pairs**.bracket The keys are also known as properties, and the values can be any type of data, including other objects, arrays, or functions. Objects are one of the fundamental building blocks in JavaScript and are used extensively in both client-side and server-side development.

Creating an Object

There are a few ways to create an object in JavaScript:

  • Using Object Literals (Most Common Way)

    • const person = {
    name: "John",
    age: 30,
    isEmployed: true
    };

  • Using the `new Object()` Syntax

    • const person = new Object();
    person.name = "John";
    person.age = 30;
    person.isEmployed = true;

    Accessing Object Properties

    You can access the properties of an object using either **dot notation** or **bracket notation**.

  • Dot Notation

    • console.log(person.name); // John
    console.log(person.age); // 30

  • Bracket Notation

    • Bracket notation is useful when the property names are dynamic or contain spaces or special characters.

    console.log(person["name"]); // John
    console.log(person["age"]); // 30

    Modifying Object Properties

    You can modify an object's properties by assigning new values:

    person.age = 31;
    console.log(person.age); // 31

    You can also add new properties:

    person.city = "New York";
    console.log(person.city); // New York

    Deleting Properties

  • You can remove a property from an object using the `delete` keyword:

    • delete person.city;
    console.log(person.city); // undefined

    Methods in Objects

  • can also contain **methods**, which are functions that are associated with the object.
    A method is defined by assigning a function to a property of the object.

    • const person = {
    name: "John",
    age: 30,
    greet: function() {
    console.log("Hello, " + this.name);
    }
    };
    person.greet(); // Hello, John

  • You can also define methods using shorthand syntax:

    • const person = {
    name: "John",
    age: 30,
    greet() {
    console.log("Hello, " + this.name);
    }
    };
    person.greet(); // Hello, John

    `this` Keyword

  • Inside an object's method, the `this` keyword refers to the object itself. For example:

    • const person = {
    name: "John",
    age: 30,
    greet() {
    console.log("Hello, " + this.name);
    }
    };
    person.greet(); // Hello, John

    ```

    Object Methods

  • `Object.keys()`
    Returns an array of the object's keys.

    • const person = { name: "John", age: 30 };
    console.log(Object.keys(person)); // ["name", "age"]

  • `Object.values()`
    Returns an array of the object's values.

    • console.log(Object.values(person)); // ["John", 30]

  • `Object.entries()`
    Returns an array of an object's key-value pairs.

    • console.log(Object.entries(person)); // [["name", "John"], ["age", 30]]

  • `Object.assign()`
    Copies the values of all properties from one or more source objects to a target object.

    • const person1 = { name: "John" };
    const person2 = { age: 30 };
    const mergedPerson = Object.assign({}, person1, person2);
    console.log(mergedPerson); // { name: "John", age: 30 }
